## 题目描述

给定正整数 $n$，请找到正整数  $l,r$，使得 $l\le \sqrt[3]{n} \lt r$，并且 $l+1=r$。

换句话说，你需要保证 $l^3\le n$ 并且 $n\lt r^3$

## 输入格式

第一行一个正整数 $n$。  

## 输出格式

一行两个正整数 $l,r$。

```input1
27
```

```output1
3 4
```

```input2
300
```

```output2
6 7
```

## 样例解释

- $3^3=27$、$4^3=64$
- $6^3=216$、$7^3=343$

## 数据规模与约定

对于 $100\%$ 的数据，$1 \le n \le 10^{18}$。

- 子任务 1（30 分）：$n\le 9$。
- 子任务 2（30 分）：$n\le 10^9$。
- 子任务 3（40 分）：没有特殊限制。
