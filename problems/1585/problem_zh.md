## 题目描述

输入 $x$，显然可以找到一个整数 $k$ 使得 $\sqrt{x} = k\sqrt{y}$。

比如当 $x=80$ 时，$\sqrt{80} = 1\sqrt{80} = 2\sqrt{20} = 4\sqrt{5}$。

请输出 $k$ 的最大值。

## 输入格式

正整数 $x$。

## 输出格式

整数 $k$。

```input1
80
```

```output1
4
```

```input2
16
```

```output2
4
```

```input3
1000
```

```output3
10
```

```input4
1
```

```output4
1
```

```input5
100000000000000
```

```output5
10000000
```

## 数据规模与约定

对于 $100\%$ 的数据，$1 \le x \le 10^{14}$。

- 子任务 1（30 分）：保证 $x$ 是一个完全平方数。
- 子任务 2（30 分）：保证 $x\le 10^9$。
- 子任务 3（40 分）：没有特殊限制。
