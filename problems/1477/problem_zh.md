## 题目描述

输入两个整数 $a,b$。

请判断 $a^b$ 与 $b^a$ 哪个更大。

- 如果 $a^b>b^a$ ，请输出：`first`
- 如果 $a^b<b^a$ ，请输出：`second`
- 如果 $a^b=b^a$ ，请输出：`same`

## 输入格式

输入空格隔开的两个整数：$a,b$

## 输出格式

如题所述，输出 `first`、`second`、`same`

```input1
2 3
```

```output1
second
```

```input2
2 4
```

```output2
same
```

```input3
4 5
```

```output3
first
```

## 数据规模与约定

对于 $100\%$ 的数据，$1 \le a,b \le 15$。

- 子任务1（30 分）：保证 $a=2$
- 子任务2（30 分）：保证 $1\le a\le 5$
- 子任务3（40 分）：没有特殊限制

有兴趣的同学可以借助自己的程序思考思考，如果数据范围更大，这题可不可以不用循环完成。
