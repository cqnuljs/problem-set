## 题目描述

给定正整数 $n$，请输出三个非负整数 $a,b,c$，使得 $a+b+c=n$。

可能有多种答案，**输出其中任意一种即可**。

## 输入格式

第一行一个整数 $n$。  

## 输出格式

一行三个整数，即 $a,b,c$。

```input1
10
```

```output1
3 3 4
```

## 样例解释

除了 $3\ 3\ 4$ 之外，输出 $1\ 2\ 7$、$4\ 1\ 5$ 等都可以。

## 数据规模与约定

对于 $100\%$ 的数据，$3 \le n \le 100$。

- 子任务 1（30 分）：$n=3$。
- 子任务 2（30 分）：保证 $n$ 是 $3$ 的倍数。
- 子任务 3（40 分）：没有特殊限制。
