## 问题陈述

有一棵树，树上有 $N$ 个顶点，编号为 $1, 2, \ldots, N$ 。对于每个 $i$ ( $1 \leq i \leq N - 1$ )， $i$ -th 边连接顶点 $x_i$ 和 $y_i$ 。

太郎决定将每个顶点涂成白色或黑色。这里不允许将相邻的两个顶点都涂成黑色。

求 $10^9 + 7$ 中可以涂抹顶点的方法的个数。

## 限制因素

- 输入值均为整数
- $1 \leq N \leq 10^5$
- $1 \leq x_i, y_i \leq N$
- 给定图形是一棵树。

## 输入

输入内容由标准输入法提供，格式如下：

- $N$
- $x_1$ $y_1$
- $x_2$ $y_2$
- $:$
- $x_{N - 1}$ $y_{N - 1}$

## 输出

打印绘制顶点的方式数，取模 $10^9 + 7$ 。

```input1
3
1 2
2 3
``` 

```output1
5
```
