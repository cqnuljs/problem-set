## 问题陈述

求在 $1$ 和 $K$ （含）之间满足以下条件的整数个数，模为 $10^9 + 7$ ：

- 十进制数位之和是 $D$ 的倍数。

## 限制因素

- 所有输入值均为整数。
- $1 \leq K \lt 10^{10000}$
- $1 \leq D \leq 100$ 

## 输入

输入内容由标准输入法提供，格式如下：

- $K$
- $D$

## 输出

打印满足条件的整数个数，取模 $10^9 + 7$ 。

```input1
30
4
``` 

```output1
6
```

```input2
1000000009
1
``` 

```output2
2
```

请务必打印出 $10^9 + 7$ 的模数。

```input3
98765432109876543210
58
``` 

```output3
635270834
```
